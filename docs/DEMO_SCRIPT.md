# Demo Script

Use this flow for a screen recording or interview walkthrough.

1. Open the landing page and explain the problem: frontend developers need API responses before backend endpoints are ready.
2. Open the Studio Dashboard.
3. Register or log in with the seeded account.
4. Create a workspace named `Demo Store`.
5. Create `GET /products` with a JSON array response.
6. Show endpoint settings for method, path, status code, delay, JSON response, and optional error simulation.
7. Click `Test` and show the mock JSON in the console.
8. Point out the workspace overview metrics for endpoints, request logs, delay, and errors.
9. Show optional API key protection for public mock endpoints.
10. Open `/docs/demo-store` and show the generated docs page.
11. Open `/api/docs/demo-store/openapi` and show the OpenAPI JSON export.
12. Return to the dashboard and show request history.
13. Mention optional local Ollama JSON generation, MySQL migrations, and CI validation.

Suggested video length: 60 to 90 seconds.
