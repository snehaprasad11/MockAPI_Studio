# Demo Script

Use this flow for a screen recording or interview walkthrough.

1. Open the landing page and explain the problem: frontend developers need API responses before backend endpoints are ready.
2. Open the Studio Dashboard.
3. Register or log in with the seeded account.
4. Create a workspace named `Demo Store`.
5. Create `GET /products` with a JSON array response.
6. Click `Test` and show the mock JSON in the console.
7. Open `/docs/demo-store` and show the generated docs page.
8. Return to the dashboard and show request history.
9. Mention optional local Ollama JSON generation.

Suggested video length: 60 to 90 seconds.
